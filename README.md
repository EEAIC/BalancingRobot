## 1. 로봇 기술
* 밸런싱 
* 음악 재생
* 비쥬얼라이져

## 2. 코드 내용
* PID 제어
  * P(비례), I(적분),D(미분)의 3개 동작을 조합시킨 제어방식
* 모터 제어
* 블루투스 연동 (RN-52 Module 사용)
* Fast Fourier Transform (라이브러리 사용)

## 3. 개발 플랫폼
* Arduino IDE
  * Arduino Uno 2 개
    * Software Serial를 통해 통신
